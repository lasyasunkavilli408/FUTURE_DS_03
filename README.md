# 📊 Marketing Funnel & Conversion Analysis

## 📌 Objective
Analyze marketing funnel data to identify conversion rates, drop-offs, and channel performance.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📂 Dataset
- Olist Marketing Funnel Dataset

## 🔍 Key Analysis
- Funnel conversion rate
- Lead source performance
- Business segment analysis

## 📈 Results
- Identified conversion percentage
- Found high drop-off in early stages
- Analyzed best performing lead sources

## 📊 Visualizations
- Funnel Conversion Chart
- Lead Source Analysis
- Segment Performance Chart

## 💡 Insights
- Only a small percentage of leads convert
- Drop-offs occur at early funnel stages
- Some channels perform better than others

## 🚀 Conclusion
Improving funnel stages can significantly increase conversions and business growth.
